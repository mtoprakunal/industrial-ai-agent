# Structured Text Kod Parçacıkları

Kopyala-yapıştır hazır ST kod parçacıkları.

## Kategoriler (eklenecek)
- Motor start/stop mantığı
- Analog ölçekleme (4-20mA → engineering unit)
- PID döngüsü
- Timer kullanımı
- OPC-UA subscription yönetimi
- Alarm oluşturma ve yönetme
- Hata yönetimi ve watchdog
