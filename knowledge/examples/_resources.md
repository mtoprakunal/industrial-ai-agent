# Öğrenme Kaynakları İndeksi

Agent'ın derinleşmek için başvurabileceği dış kaynakların kürasyonlu listesi. Her
kaynak `knowledge/examples/` altındaki ilgili kayıtta daha ayrıntılı ele alınmıştır.
Kod/dosya bu repoda saklanmaz; resmi kaynaktan (lisansına uyularak) alınır.

## Açık Kaynak Kütüphaneler / Projeler

- **OSCAT** — http://www.oscat.de — BASIC/NETWORK/BUILDING IEC 61131-3 kütüphaneleri.
  CODESYS Store üzerinden de dağıtılır. (Kendi özel lisansı; atıf zorunlu.)
- **CODESYS Forge** — https://forge.codesys.com/prj/ — açık kaynak topluluk projeleri.
- **CODESYS Store** — https://store.codesys.com — resmi + ücretsiz örnekler.
- **awesome-structured-text** — https://github.com/myutzy/awesome-structured-text
- **OpenPLC** — https://www.openplcproject.com — açık kaynak PLC runtime/editör.
- **RuSTy (PLC-lang)** — https://github.com/PLC-lang/rusty — ST derleyici.
- **TcUnit** — https://tcunit.org / https://github.com/tcunit/TcUnit — TwinCAT birim test.

## Vendor Örnekleri / Uygulama Notları

- **CODESYS Online Help — Examples** — https://help.codesys.com
- **Beckhoff InfoSys + GitHub Samples** — https://infosys.beckhoff.com ,
  https://github.com/Beckhoff (TF... örnek depoları)
- **Inovance Downloads** — https://www.inovance.eu/downloads (AM600/InoProShop örnekleri)

## Referans Mimariler / Standartlar

- **ISA-95 / IEC 62264** — https://www.isa.org/isa95
- **Eclipse Sparkplug** — https://sparkplug.eclipse.org ; **Eclipse Tahu** —
  https://github.com/eclipse/tahu (UNS / MQTT Sparkplug B)
- **OPC-UA Companion Specs** — https://reference.opcfoundation.org
- **NAMUR (NOA, NE serisi)** — https://www.namur.net

## PLCopen / Motion

- **PLCopen** — https://plcopen.org (motion FB standardı, XML)
- **CODESYS SoftMotion** dokümantasyonu — https://help.codesys.com
