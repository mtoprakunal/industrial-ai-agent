# PLCopen XML Şablonları

IEC 61131-10 standardına uygun PLCopen XML şablonları.
CODESYS'e doğrudan import edilebilir.

## İçe Aktarma
File > Import PLCopen XML

## Mevcut Şablonlar (eklenecek)
- `basic_program.xml`      — Temel program yapısı
- `motor_control_fb.xml`   — Motor kontrol fonksiyon bloğu
- `analog_scaling_fb.xml`  — Analog ölçekleme fonksiyon bloğu
- `alarm_handler_fb.xml`   — Alarm yönetim fonksiyon bloğu
