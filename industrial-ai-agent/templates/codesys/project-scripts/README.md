# CODESYS Script Engine Şablonları

Bu klasör, CODESYS Script Engine (IronPython) ile
projeyi otomatik dolduran Python scriptlerini içerir.

## Nasıl Kullanılır
1. CODESYS'te yeni boş proje aç
2. Tools > Scripting > Execute Script File
3. İlgili scripti seç
4. Script projeyi otomatik yapılandırır

## Mevcut Scriptler (eklenecek)
- `create_basic_structure.py` — Temel task, GVL yapısı
- `add_opcua_server.py`       — OPC-UA sunucu konfigürasyonu
- `add_modbus_slave.py`       — Modbus slave konfigürasyonu
- `generate_gvl_from_io.py`   — I/O listesinden GVL üretimi
