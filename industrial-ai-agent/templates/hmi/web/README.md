# Web Tabanlı HMI Şablonları

Farklı JavaScript/web framework'leri için
OPC-UA ve Modbus TCP istemci şablonları.

## Eklenecek Şablonlar
- `opcua-react/`    — React + node-opcua başlangıç şablonu
- `opcua-vue/`      — Vue.js + opcua-client başlangıç şablonu
- `modbus-react/`   — React + jsmodbus başlangıç şablonu
- `vanilla-js/`     — Framework bağımsız temel şablon

## Her Şablon İçerir
- OPC-UA/Modbus bağlantı yönetimi
- Gerçek zamanlı veri görüntüleme
- Alarm paneli
- Tag yazma (output kontrol)
- Bağlantı durumu göstergesi
